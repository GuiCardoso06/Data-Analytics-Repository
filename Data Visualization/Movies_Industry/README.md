This project was developed as part of the **Data Visualization (LECD) 2023/2024** course and focuses on exploring and communicating insights from a movie-related dataset through interactive and well-designed visualizations.

The main objective is to analyze relationships between different movie attributes and present them in a clear and intuitive way using multiple visualization techniques.
